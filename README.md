# Shopping Cart App

### [Odeane Croney](https://github.com/Odeane)

## General info
This project is a simple shopping cart that allows users to add items and display the total cost.

## Dependencies

- npm
- React
- React-redux
- redux
- redux-thunk
- axios

## Setup

To run this project, install it locally using npm:

```
  $ npm install
  $ npm start
```
